Doubly Linked List in C

Complete In-Depth Breakdown & Visualizations

1. Defining the Node Structure

Before writing functions, we must define what a "Node" looks like in our computer's memory. Unlike a singly linked list, a Doubly Linked List node has three distinct parts.

Blueprint of a Node
prev
data
next

We use a struct because a node contains different types of data grouped together. We also declare a global pointer called head.

struct node {
    int data;               // Stores the actual value
    struct node *next;      // Pointer to the next node
    struct node *prev;      // Pointer to the previous node
};

struct node *head = 0;      // Global pointer, initially NULL
Instructor Note: head is declared globally (outside of any function) because we need both our create() function and our display() function to easily access the starting point of our list.

2. Allocating Memory & Taking Input

To create a node, we must ask the computer for memory while the program is running using malloc().

void create() {
    struct node *newnode; 
    
    // 1. Allocate memory dynamically
    newnode = (struct node *)malloc(sizeof(struct node));

    // 2. Populate the node
    printf("Enter data: ");
    scanf("%d", &newnode->data);
    newnode->prev = 0;
    newnode->next = 0;
Memory State: After User enters '7'

0
Address: 250

At this exact moment, the new node is floating alone in memory. It isn't connected to anything yet, so its prev and next pointers must point to 0 (NULL).

3. Case 1: Inserting the First Node

How do we know if this is the very first node being created? The global head pointer will be 0.

    if (head == 0) {
        head = temp = newnode;
    }
Memory State: First Node Inserted

Address: 250
temp (250)

Why do we need temp? head must always stay pointing at the first node. We use temp as a moving pointer to keep track of the last node in the list, so we know where to attach the next new node.

4. Case 2: Adding Subsequent Nodes

What happens when the user wants to add another node (e.g., Data '1')? head is no longer 0, so we move to the else block.

We must update two links (forward and backward) and then move our temp pointer to the new end of the list.

    else {
        temp->next = newnode;   // Link 1: Connect current last node forward
        newnode->prev = temp;   // Link 2: Connect new node backward
        temp = newnode;         // Move temp to the new end
    }

Instructor Warning: Do not write head->next = newnode here! While it works for the 2nd node, it fails when you add a 3rd or 4th node because head is stuck at the beginning. This is why we must use temp to manage the end of the list.

5. Displaying the List

Traversing a Doubly Linked List is exactly the same as traversing a Singly Linked List.

void display() {
    struct node *temp;      
    temp = head;            // Start at the beginning
    
    while (temp != 0) {             // Keep going until we hit NULL
        printf("%d ", temp->data);  
        temp = temp->next;          // Hop to the next node
    }
}
Golden Rule: Never move the head pointer to traverse the list! If you write head = head->next, you lose the location of the first node, permanently losing your list in memory. Always use a local temp pointer for traveling.

6. The Complete C Program

Here is the complete, interactive logic written by the instructor on the whiteboard, compiled into a single copyable program.

#include <stdio.h>
#include <stdlib.h>

struct node {
    int data;
    struct node *next;
    struct node *prev;
};

struct node *head = 0;

void create() {
    struct node *newnode, *temp;
    int choice = 1;
    
    while (choice == 1) { 
        newnode = (struct node *)malloc(sizeof(struct node));
        
        printf("Enter data: ");
        scanf("%d", &newnode->data);
        newnode->prev = 0;
        newnode->next = 0;
        
        if (head == 0) {
            head = temp = newnode; 
        } 
        else {
            temp->next = newnode;  
            newnode->prev = temp;  
            temp = newnode;        
        }
        
        printf("Do you want to continue? (1 Yes, 0 No): ");
        scanf("%d", &choice);
    }
}

void display() {
    struct node *temp;
    temp = head; 
    
    printf("\nThe Doubly Linked List is: ");
    while (temp != 0) {
        printf("%d ", temp->data); 
        temp = temp->next;         
    }
    printf("\n");
}

int main() { 
    create();
    display();
    return 0;
}
