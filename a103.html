Heap Sort Breakdown

An in-depth, vertical scroll masterclass. Learn how to sort arrays in-place using Complete Binary Trees, starting from basic insertion to the highly optimized Heapify method.

The Big Picture

Heap Sort is an in-place, comparison-based sorting algorithm. To sort an array in ascending order, we use a Max Heap. There are always exactly TWO main steps to Heap Sort:

1

Build a Max Heap

Rearrange the given array into a valid Max Heap structure where every parent is $\ge$ its children.

2

Delete Elements

Repeatedly swap the largest element (the root) to the end of the array, shrink the heap size, and restore the heap.

Method 1: Building via Insertion

The intuitive way to build a heap is to insert elements one by one into a Complete Binary Tree, swapping upwards if a child is larger than its parent. (Note: We will learn a faster way later!)

Optimization Alert 🔥

The `Heapify` Method

Building the heap via insertion takes $O(n \log n)$. We can build the initial heap in $O(n)$ time by using the Bottom-Up Heapify approach.

The Secret Math Trick

In a complete binary tree of $n$ elements, the leaf nodes are located from index floor(n/2) + 1 to n.

Leaf nodes are already valid heaps. We only need to run the `Heapify` function on non-leaf nodes, starting from index floor(n/2) backwards to 1.

The Final Code

1. MaxHeapify Function

void MaxHeapify(int A[], int n, int i) {
    int largest = i;          // Initialize largest as root
    int l = (2 * i);          // Left child
    int r = (2 * i) + 1;      // Right child

    // If left child is larger than root
    if (l <= n && A[l] > A[largest])
        largest = l;

    // If right child is larger than largest so far
    if (r <= n && A[r] > A[largest])
        largest = r;

    // If largest is not root
    if (largest != i) {
        swap(A[i], A[largest]);
        MaxHeapify(A, n, largest); // Recursively heapify
    }
}

2. Main HeapSort Function

void heapSort(int A[], int n) {
    // STEP 1: Build Max Heap (O(n) time)
    for (int i = n / 2; i >= 1; i--) {
        MaxHeapify(A, n, i);
    }

    // STEP 2: Delete elements to sort (O(n log n) time)
    for (int i = n; i >= 1; i--) {
        swap(A[1], A[i]);        // Move current root to end
        MaxHeapify(A, i - 1, 1); // Heapify reduced heap
    }
}
