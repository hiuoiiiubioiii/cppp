The Core Rules

The instructor explains that the name of the traversal tells you exactly where the Root goes. Memorize these formulas:

Instructor's Tip:
  • INorder: The root is IN the middle.
  • PREorder: The root is PRE (before) everything else.
  • POSTorder: The root is POST (after) everything else.

1. Inorder Traversal

Formula: Left ➔ Root ➔ Right

Starting at the very top of the tree (Root A), we follow the rule rigidly for every single node.

  1. Start at A. Rule says go Left. Go to B.
  2. At B, rule says go Left. There is nothing (Null) to the left of B.
  3. Since Left is done, we print the Root of this sub-tree: Write B.
  4. Rule says go Right. Go to D. Write D. (D has no children, so it's done).
  5. Backtrack to A. The entire left side of A is now done. Next step in the rule is Root. Write A.
  6. Rule says go Right. Go to C.
  7. At C, rule says go Left. Go to E.
  8. At E, rule says go Left. Go to G.
  9. At G, there is no Left. Print the root: Write G.
  10. Backtrack to E. Left is done, so print the root: Write E. (E has no right child).
  11. Backtrack to C. Left side of C is done. Print the root: Write C.
  12. Rule says go Right. Go to F.
  13. At F, rule says go Left. Go to H. Write H.
  14. Backtrack to F. Left is done, print the root: Write F.
  15. Rule says go Right. Go to I. Write I.
Final Inorder:
B, D, A, G, E, C, H, F, I

2. Preorder Traversal

Formula: Root ➔ Left ➔ Right

Because the Root comes first, you write the node down as soon as you visit it for the first time.

  1. Start at the top. Print the root immediately: Write A.
  2. Rule says go Left. Go to B. Print the root: Write B.
  3. Rule says go Left. It's Null. Go Right. Go to D. Print the root: Write D.
  4. The entire left side of A is done. Backtrack to A and go Right to C.
  5. Print the root: Write C.
  6. Rule says go Left. Go to E. Print the root: Write E.
  7. Rule says go Left. Go to G. Print the root: Write G.
  8. Backtrack up to C, then go Right. Go to F. Print the root: Write F.
  9. Rule says go Left. Go to H. Print the root: Write H.
  10. Backtrack to F, go Right. Go to I. Print the root: Write I.
Final Preorder:
A, B, D, C, E, G, F, H, I

3. Postorder Traversal

Formula: Left ➔ Right ➔ Root

The root is printed last. You can only write a node down when you have fully checked its left and right children.

  1. Start at A. Go Left to B.
  2. At B, go Left (Null). Go Right to D.
  3. D has no children. Left and Right are done. Print root: Write D.
  4. Backtrack to B. Left and Right are now done. Print root: Write B.
  5. Backtrack to A. The whole left side is done. Go Right to C.
  6. At C, go Left to E.
  7. At E, go Left to G. Left/Right of G are null. Print root: Write G.
  8. Backtrack to E. Right is null. Left/Right are done. Print root: Write E.
  9. Backtrack to C. Left is done. Go Right to F.
  10. At F, go Left to H. Left/Right of H are null. Print root: Write H.
  11. Backtrack to F. Go Right to I. Left/Right of I are null. Print root: Write I.
  12. Backtrack to F. Left and Right are done. Print root: Write F.
  13. Backtrack to C. Left and Right are done. Print root: Write C.
  14. Backtrack to A. The entire Left and Right sides of the tree are done. Finally, print the ultimate root: Write A.
Final Postorder:
D, B, G, E, H, I, F, C, A
