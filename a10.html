Implementation of a Linked List

Continuous Scroll Study Guide • Based on Instructor's Whiteboard Breakdown

1. The Logical View (Whiteboard)

Before writing code, we need to understand what a Singly Linked List looks like conceptually. It is a chain of nodes. Each node has two parts: the data (actual value) and the address of the next node.

We must maintain a head pointer, which always stores the address of the very first node. The last node points to 0 (NULL).

Instructor's Golden Rule:

Never move the head pointer during traversal or insertion. If you lose the head pointer, you lose access to your entire linked list!

2. Defining the Blueprint (Struct)

To create a node that can hold both an integer and a pointer to another node, we must create a custom data type in C using the struct keyword.

struct node {
    int data;
    struct node *next;
};

Why `struct node *next`?

The `next` part is a pointer. It stores a memory address. Because it stores the address of another node of this exact same structure, its data type must be a pointer to `struct node`.

3. Declaring Pointers & Initialization

To build the list, we need three specific pointers in our program.

struct node *head, *newnode, *temp;
head = 0; // Same as NULL
  • head: Will permanently store the address of the first node.
  • newnode: Will store the address of the node we are currently creating.
  • temp: A temporary pointer used to traverse the list and link new nodes without moving `head`.

Initially, we set `head = 0` because our list is empty. There are no nodes yet.

4. Dynamic Memory Allocation (malloc)

Nodes aren't created automatically. We have to request memory dynamically while the program is running using the malloc() function.

newnode = (struct node *)malloc(sizeof(struct node));

Breaking down this line:

  1. sizeof(struct node) calculates how many bytes we need. (e.g., 4 bytes for int + 4 bytes for pointer = 8 bytes).
  2. malloc(8) finds an 8-byte block in memory and returns a void pointer to its starting address.
  3. (struct node *) is Typecasting. It tells the compiler, "Treat this generic void pointer as a pointer to a node structure."
  4. Finally, that starting address is stored in newnode.

5. Inserting Data (The Arrow Operator)

Once memory is allocated, we need to put data into it. Because `newnode` is a pointer, we cannot use the standard dot (`.`) operator to access the struct members. We must use the -> (arrow) operator.

printf("Enter data: ");
scanf("%d", &newnode->data);
newnode->next = 0; // Initialize next as NULL

At the moment of creation, the new node isn't attached to anything, so its `next` pointer should safely point to `0`.

6. The Core Logic: Linking the Nodes

When attaching a node, we have two scenarios: is it the first node ever created, or is it a subsequent node?

if (head == 0) {
    head = temp = newnode;
} else {
    temp->next = newnode;
    temp = newnode;
}

Visualizing the Logic

If `head == 0` (First Node): The list is empty. Both `head` and `temp` point to this brand new node.

7. Traversing & Printing the List

Once the user stops adding nodes, we want to print them all out. We reset our `temp` pointer to the `head` and loop until we hit the end (`0`).

temp = head; // Reset temp to start
while (temp != 0) {
    printf("%d ", temp->data);
    temp = temp->next; // Move to the next node
}

Understanding `temp = temp->next;`

This is the standard line to traverse a linked list. It literally translates to: "Take the address stored in the current node's 'next' block, and make that my new current location."

8. The Complete Source Code

Here is the full, combined C program written by the instructor on the whiteboard.

#include <stdio.h>
#include <stdlib.h>

// Define the structure
struct node {
    int data;
    struct node *next;
};

void main() {
    struct node *head, *newnode, *temp;
    head = 0; 
    int choice = 1;

    // Node Creation Loop
    while (choice) {
        newnode = (struct node *)malloc(sizeof(struct node));
        
        printf("Enter data: ");
        scanf("%d", &newnode->data);
        newnode->next = 0;
        
        if (head == 0) {
            head = temp = newnode;
        } else {
            temp->next = newnode;
            temp = newnode;
        }
        
        printf("Do you want to continue (0,1)? ");
        scanf("%d", &choice);
    }
    
    // Traversal and Printing
    temp = head;
    printf("\nLinked List data: ");
    while (temp != 0) {
        printf("%d ", temp->data);
        temp = temp->next;
    }
    
    getch();
}
