Kruskal's Algorithm

Finding the Minimum Spanning Tree (MST) | Instructor: Jenny's Lectures

1. Introduction to the Algorithm

The goal of Kruskal's Algorithm is to find the Minimum Spanning Tree (MST) of a given weighted, undirected graph.

A Spanning Tree is a subgraph that connects all the vertices together without any cycles. A Minimum Spanning Tree is the one where the sum of all edge weights is as small as possible.

Context: There are two main algorithms used to find an MST: Kruskal’s Algorithm and Prim’s Algorithm. This guide focuses entirely on Kruskal's.

3. Step 1: Simplify the Graph

Before applying the algorithm, you must clean up the graph by removing two things:

  • Parallel Edges: Two edges connecting the exact same pair of vertices. Rule: Always keep the edge with the minimum weight and delete the larger one. (Removed A-B edge with weight 10).
  • Loops: An edge that starts and ends on the exact same vertex. Rule: Remove loops entirely. (Removed loop at E with weight 1).

5. Step 3: Construct the MST

Go down your sorted list from top to bottom. Choose the edge with the minimum weight and draw it.

Crucial Condition: You must NOT include an edge if drawing it would create a closed loop (a cycle) in your new spanning tree.

Evaluating the Edges:

  • BD (2): Added. No cycle.
  • DE (2): Added. No cycle.
  • AC (3): Added. No cycle (it's okay if parts of the tree are temporarily disconnected).
  • CD (3): Added. No cycle.

Handling Cycles (Rejections):

  • BE (4): REJECTED. Creates triangle cycle (B → D → E → B).
  • AB (5): REJECTED. Creates large cycle (A → C → D → B → A).
  • BC (6): REJECTED. Creates cycle (B → D → C → B).

Finishing the Tree:

  • AF (7): Added. No cycle formed.
  • FC (8): REJECTED. Creates cycle (F → A → C → F).

6. Verifying the Final MST

The instructor outlines two mathematical properties to check if your MST is correct and complete:

Property 1: No Cycles
Visually verify that the final drawing does not contain any closed loops.
Property 2: Vertex to Edge Ratio
If the original graph has n vertices, the MST must have exactly n vertices and n - 1 edges.
