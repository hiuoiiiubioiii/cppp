Step 2: Connect the Pointers

If the list is empty (head == 0), the new node becomes both head and tail, and points to itself. Otherwise, we execute the following steps to link it to our baseline list.

Step A: newnode->next = head;
The new node (500) needs to point forward to the old first node (200).
Step B: head->prev = newnode;
The old first node (200) needs its prev to point back to the new node (500).
Step C: newnode->prev = tail;
Because it's circular, the new first node (500) must have its prev point to the very last node (100).
Step D: tail->next = newnode;
The last node (100) must have its next point to the new first node (500).
Step E: head = newnode;
Shift the head pointer to recognize 500 as the official start.

⚠️ Instructor Warning: Order Matters!

Look at Step E. You must update head = newnode; at the very end. If you move the head pointer first, you lose your only reference to address 200, and you will break the rest of the list!

4. Insertion at a Specific Position

Goal: Traverse the list and insert a node exactly where the user requests (e.g., Position 3).

Step 1: Setup and Validation

void insertAtPos() {
    struct node *newnode, *temp;
    int pos, i = 1, l;
    
    printf("Enter position: ");
    scanf("%d", &pos);
    
    l = getLength(); // Custom func to count nodes
    
    if(pos < 1 || pos > l) {
        printf("Invalid position");
    }
    else if(pos == 1) {
        insertAtBeg(); // Reuse existing logic!
    }

Step 2: Traverse and Insert

If the user enters pos = 3, we need to stop at position 2. We use a while loop running while i < pos - 1.

    else {
        // ... [Allocate memory for newnode (Addr: 400, Data: -1)] ...
        
        temp = head;
        while(i < pos - 1) {
            temp = temp->next;
            i++;
        }
        
        // Update Pointers
        newnode->prev = temp;                 // Step A
        newnode->next = temp->next;           // Step B
        temp->next->prev = newnode;           // Step C
        temp->next = newnode;                 // Step D
    }
}
The Traversal: temp stops at Node 2 (address 300). We are inserting 400 between 300 and 100.
Step A & B: Set up the new node first. newnode->prev = temp (points to 300). newnode->next = temp->next (points to 100).
Step C: Update the node after. temp->next->prev = newnode (Node 100's prev points to 400).
Step D: Update the node before. temp->next = newnode (Node 300's next points to 400).

⚠️ Instructor Warning: Protect the Chain!

Always set the next and prev of your newly created node first (Steps A & B). If you accidentally overwrite temp->next too early, you break the chain and completely lose access to the rest of your list!
