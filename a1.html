Arrays in Data Structure

Complete Masterclass Notes & Visualizations

1. Why do we need Arrays?

Standard variables can only hold one value at a time. If you try to store a new value, the old one is overwritten.

int a = 5;  // Memory stores 5
a = 6;      // 5 is now deleted/overwritten by 6
The Array was created to store large amounts of data (e.g., 60 student roll numbers) under a single variable name without declaring 60 different variables.

2. Memory Representation

Computer memory is viewed as a long tape of bytes. Each byte consists of 8 bits.

  • 1 Integer (int): Typically consumes 4 Bytes (32 bits).
  • Binary Conversion: The value 5 is stored as 32 binary digits.

3. Defining an Array

An array is a fixed-size, sequenced collection of data items of the same data type.

Data_Type Array_Name[Size];

Validation Rules

Declaration Status Reason
int a[5] = {1, 2, 3, 4, 5}; VALID Homogeneous (All integers)
char b[10]; VALID Standard character array
int a[]; INVALID Size must be specified
int a[5] = {1, 'j', 3}; INVALID Heterogeneous (Mixed types)

5. Initialization Methods

A. Compile-Time (Static)

Values are assigned during declaration.

int a[5] = {6, 2, 4, 3, 0};

B. Run-Time (Dynamic)

Asking the user for input while the program is running using a loop.

for(int i=0; i < 5; i++) {
    scanf("%d", &a[i]);
}

6. The Access Formula

Arrays support Random Access. You can calculate the address of any element instantly using the formula:

Address = Base + (Index * Size)

Example Calculation:
To find a[2] where Base = 100 and Size = 4 bytes:
100 + (2 * 4) = 108

  • Time Complexity: O(1) (Constant Time)
  • Index: Always starts at 0.

7. Drawbacks of Arrays

While powerful, arrays have significant limitations:

  1. Fixed Size: You cannot increase or decrease the size at runtime.
  2. Wastage of Space: If you declare 100 slots but use only 10, the other 90 slots remain occupied and empty.
  3. Tedious Re-allocation: To expand an array, you must create a new larger array and manually copy all elements from the old one.
