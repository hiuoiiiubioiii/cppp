1. Introduction to Binary Trees

A binary tree is a specific type of tree data structure where each node can have a maximum of two children.

  • "Binary" means 2. Therefore, a node can have exactly 0, 1, or 2 children.
  • It cannot have 3 or more children.

2. Memory Representation

To store a binary tree in computer memory, we use a dynamic linked representation. Every node consists of three distinct parts:

Left Ptr
Data (Value)
Right Ptr
  • Left Pointer: Stores the memory address of the left child.
  • Data: Stores the actual value/information.
  • Right Pointer: Stores the memory address of the right child.
  • If a child doesn't exist (like in a leaf node), the pointer holds Null (X).

3. Mathematical Properties

Deriving formulas based on the structure of binary trees.

A. Maximum Nodes at a Specific Level (i)

If you pick any level of the tree, what is the maximum number of nodes that can fit on that line?

Max nodes at level i = 2i
  • Level 0: 20 = 1 node
  • Level 1: 21 = 2 nodes
  • Level 2: 22 = 4 nodes

B. Maximum Nodes for a Tree of Height (h)

Adding together the max nodes at every single level from 0 up to h forms a Geometric Progression.

Max total nodes (n) = 2h+1 - 1

C. Minimum Nodes for a Tree of Height (h)

What is the bare minimum number of nodes required to reach height h? You only need one node per level (a straight line).

Min total nodes (n) = h + 1

D. Finding Height (h) when given Nodes (n)

Reverse the formulas above to find the height:

Minimum Height: h = ⌈ log2(n+1) ⌉ - 1

Maximum Height: h = n - 1

4. Types of Binary Trees

Type 1: Full / Proper / Strict Binary Tree

Rule: Each node has either 0 or 2 children. (No node is allowed to have exactly 1 child).

Property: Leaf Nodes (L) = Internal Nodes (I) + 1

✅ Valid (0 or 2)

❌ Invalid (has 1 child)

Type 2: Complete Binary Tree

Rule:

  1. All levels are completely filled (except possibly the last level).
  2. The last level has nodes packed as left as possible.

✅ Valid (Left-packed)

❌ Invalid (Gap on left)

Type 3: Perfect Binary Tree

Rule: All internal nodes have exactly 2 children AND all leaves are at the exact same level. (Forms a perfect symmetrical triangle).

Note: Every Perfect Binary Tree is automatically Full and Complete, but the reverse is not true.

✅ Perfect Symmetrical Triangle
