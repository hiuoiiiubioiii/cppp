Insertion in a Circular Linked List

Comprehensive study notes featuring simulated whiteboard visualizations. Scroll down to learn.

📌 1. Initial Setup & Concepts

Unlike a singly linked list, a Circular Linked List does not have a NULL pointer at the end. Instead, the last node points back to the first node.

Instructor's Golden Rule: We implement the list using only a tail pointer (pointing to the last node), rather than a head pointer.

  • Why? With a tail pointer, you have access to both the last node (tail) and the first node (tail->next) in O(1) time!

The Node Structure

struct node {
    int data;
    struct node *next;
};
struct node *tail; // Global tail pointer

🚀 2. Insertion at the Beginning

Goal: Create a new node and make it the first node in the list.

The Logic:

If the list is not empty, the new node needs to be inserted before Node 1.

  1. Step 1: Make the new node point to the current first node. We find the first node using tail->next.
  2. Step 2: Make the last node (tail) point to the new node, because the new node is now the first node.
void insertAtBeg() {
    // 1. Create Node (Data: 7, Address: 600)
    struct node *newnode = malloc(sizeof(struct node));
    newnode->data = 7; 
    newnode->next = 0;

    if(tail == 0) {
        tail = newnode;
        tail->next = newnode; // Points to itself
    } else {
        newnode->next = tail->next; // Step 1
        tail->next = newnode;       // Step 2
    }
}

🏁 3. Insertion at the End

Goal: Create a new node and place it at the very end of the list, making it the new tail.

Crucial Warning: The order of operations matters immensely here. If you move the tail pointer first, you will lose the address of the first node!

  1. Step 1: The new node will be the new last node, so it must point back to the first node (tail->next).
  2. Step 2: The current last node (tail) must point to the new node.
  3. Step 3: Update the global tail pointer to point to the new node.
void insertAtEnd() {
    // 1. Create Node (Data: 10, Address: 700)
    struct node *newnode = malloc(sizeof(struct node));
    newnode->data = 10; 
    newnode->next = 0;

    if(tail == 0) { ... }
    else {
        newnode->next = tail->next; // Step 1
        tail->next = newnode;       // Step 2
        tail = newnode;             // Step 3: Shift tail
    }
}

🎯 4. Insertion at a Specific Position

Goal: Insert a node at a specific position (e.g., Position 3). To insert at position pos, we must stop our traversal at position pos - 1.

void insertAtPos() {
    int pos = 3, i = 1;
    // ... Validation checks for pos < 0 or pos == 1 ...
    
    struct node *newnode = malloc(...); // Data 7, Addr 250
    struct node *temp = tail->next; // Start at node 1

    // Traverse to node JUST BEFORE insertion point
    while(i < pos - 1) {
        temp = temp->next;
        i++;
    }

    // Pointer magic
    newnode->next = temp->next; // Step 1
    temp->next = newnode;       // Step 2
}

💡 Summary: Golden Rules

  • The power of tail: tail gives you the last node. tail->next ALWAYS gives you the first node.
  • Order of Pointer Updates: When inserting, always update the new node's pointer first. If you update the existing list's pointers first, you break the chain.
  • Position Traversal Rule: To insert at position N, you must stop your temp pointer at position N-1.
