1. Intro to Graphs & Spanning Trees

Before jumping into Minimum Spanning Trees, the instructor starts by defining a basic Spanning Tree using a 5-vertex pentagon graph.

Graph Definition: A standard graph is represented as G(V, E)

  • V = Number of Vertices (Nodes)
  • E = Number of Edges (Lines connecting nodes)

A Spanning Tree of that graph is represented as G'(V', E'). To be a valid spanning tree, it must obey three strict mathematical rules:

  • Rule 1: V' = V (The tree must include EVERY vertex from the original graph).
  • Rule 2: E' ⊂ E (The edges in the tree must be a subset of the original edges).
  • Rule 3 (The Formula): |E'| = |V| - 1 (The number of edges in the tree is exactly the number of vertices minus one).
Original Graph G(V, E)
Vertices = 5, Edges = 5
Valid Spanning Tree G'(V', E')
Vertices = 5, Edges = 4 (V-1)

Note: A single graph can have multiple valid spanning trees depending on which edge you choose to remove.

2. What is an MST?

The instructor introduces Edge Weights (costs associated with traveling from one vertex to another). A Minimum Spanning Tree (MST) is simply the spanning tree where the total sum of all edge weights is the lowest possible.

The MST
Total Cost = 10 (1+4+2+3)

To find this, the instructor systematically keeps the lowest weight edges (1, 2, 3, and 4) while dropping the highest (5) to meet the rule of having exactly 4 edges (V-1).

Golden Rules of MSTs:

  • No Cycles: A spanning tree must never contain a closed loop.
  • Must be Connected: Every vertex must be reachable. If a vertex is left floating, it violates the rule that V' = V.

3. Seven Essential Properties

The instructor erased the board to list out these 7 highly-testable properties of Spanning Trees and MSTs. Commit these to memory!

  • 1. Disconnection: Removing even just one edge from a valid spanning tree will make it disconnected. It breaks the tree entirely.
  • 2. Loops: Adding one edge to a valid spanning tree will immediately create a loop (cycle), invalidating the tree.
  • 3. Uniqueness: If every edge in a graph has a distinct (unique) weight, there will be exactly one unique MST. If edges share the same weights, multiple MSTs with the same minimum cost can exist.
  • 4. Complete Graphs: A complete undirected graph (where every vertex connects to every other vertex) has exactly nn-2 possible spanning trees.
  • 5. Existence: Every connected, undirected graph has at least one spanning tree.
  • 6. Disconnected Graphs: A graph that is already disconnected does not have any spanning trees.
  • 7. Edge Removal: From a complete graph, you construct a spanning tree by removing a maximum of (e - n + 1) edges.

4. Deep Dive: Edge Removal Formula

To finish the lecture, the instructor proves Property #7 using visual examples.

Maximum edges to remove = e - n + 1

(Where e = total edges, and n = total vertices).
