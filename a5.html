Topic 1

The Mental Model: What is a 2D Array?

While a 1D array is a simple list (a single line of items), a 2D array is best visualized as a Matrix or a grid. However, the core programming concept is that a 2D array is an "Array of Arrays."

Topic 2

Declaration & Memory Size

To create a 2D array, specify the data type, name, and the two dimensions in separate square brackets.

int a[3][4];
  • int : Data type (each item is an integer).
  • [3] : First subscript = Number of Rows.
  • [4] : Second subscript = Number of Columns.

Calculating Memory Size

Total Elements = Rows × Columns = 3 × 4 = 12

Size of 1 integer = 4 bytes

Total Array Size = 12 × 4 = 48 bytes

Topic 3

Initialization

A. Compile-Time Initialization

Assigning values directly in the code. Nested braces are recommended for visual clarity.

// 1. Flat list (Valid)
int a[2][3] = {1, 2, 3, 4, 5, 6};

// 2. Nested braces (Recommended)
int a[2][3] = {{1, 2, 3}, {4, 5, 6}};

// 3. Omitting row size (Valid)
int a[][3] = {{1, 2, 3}, {4, 5, 6}};

Strict Compiler Rule

int a[2][] = {1, 2, 3, 4, 5, 6}; // INVALID

You can leave the 1st subscript (rows) empty, but you MUST provide the 2nd subscript (columns). The compiler needs the column size to know exactly where one row ends and the next begins to structure the memory.

B. Run-Time Initialization (User Input)

To iterate through a 2D array, you must use nested for loops.

int a[2][3];
int i, j;

printf("Enter the elements: ");

// Outer loop controls Rows
for(i = 0; i < 2; i++) {       
    // Inner loop controls Columns
    for(j = 0; j < 3; j++) {   
        scanf("%d", &a[i][j]); 
    }
}

Exam Critical

Address Calculation Formulas

The compiler uses O(1) math formulas to instantly jump to any element's memory address without searching.

B = Base Address
size = Bytes per element (Word size)
m = Total Rows
n = Total Columns
i = Target Row Index
j = Target Column Index

A. Row-Major Formula

Logic: Skip 'i' complete rows (where each row has 'n' elements). Then step into current row and skip 'j' elements.

0-Indexed Arrays (Standard C)

Address(a[i][j]) = B + [ (i * n) + j ] * size

1-Indexed Arrays (Math/Other Languages)

Address(a[i][j]) = B + [ ((i-1) * n) + (j-1) ] * size

B. Column-Major Formula

Logic: Skip 'j' complete columns (where each column has 'm' elements). Then step down into the current column by 'i' elements.

0-Indexed Arrays

Address(a[i][j]) = B + [ (j * m) + i ] * size

1-Indexed Arrays

Address(a[i][j]) = B + [ ((j-1) * m) + (i-1) ] * size
