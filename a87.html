Detect Cycle in Directed Graph

Using Depth-First Search (DFS) | Scroll down to learn

1. Introduction & Goal

Objective: Write an algorithm that traverses a directed graph using DFS. The algorithm should return True if a cycle exists, and False if it does not.

A cycle occurs when you can follow a path of directed edges and end up back at a vertex you are currently exploring.

2. The Example Graph

The instructor draws a directed graph with 5 vertices. Notice how D points to E, and E points back to B, creating a cycle.

(A) -----> (B) -----> (D) | | | | | | v v v (C) (C) (E) | | (Loop Edge) +-----> (B)

3. Data Structures & Setup

To run this algorithm, we set up specific data structures on our whiteboard:

  • Stack: Keeps track of the current path of nodes we are exploring.
  • Visited Set: Tracks all nodes we have touched.
  • Parent Map: Remembers where we came from (Vertex | Parent) to trace the cycle later.
  • Flag Variable Array (Crucial): Every node gets a status flag.
-1 : Unvisited (Not touched yet)
0 : Visited & In Stack (Currently exploring)
1 : Visited & Popped (Fully explored, dead end reached)

Initial state: All nodes (A, B, C, D, E) start with a flag of -1.

5. The Cycle Detection Check!

We are currently at node E. We look at E's adjacent node: B.

We check the flag array for B. The flag of B is 0 (Visited & In Stack).

THE GOLDEN RULE:
If, during traversal, any vertex finds an adjacent vertex with a flag of 0 (meaning the node is already visited AND is currently sitting in the active stack), the graph contains a cycle. Return True!

6. Tracing and Printing the Cycle

Once the cycle is detected at the edge E → B, we use the Parent Map table to trace the loop backwards to print it.

  1. We know the cycle closes from E to B.
  2. Look at Parent Map for E: Parent is D.
  3. Look at Parent Map for D: Parent is B.
  4. We have reached B, which is where the loop closed.

The Cycle is: [ B → D → E → B ]

7. Time Complexity

O(V + E)

V = Number of Vertices | E = Number of Edges

Why? In the worst-case scenario (a graph with no cycles), the DFS algorithm will visit every single vertex and traverse every single edge exactly once before the stack fully empties, returning False.
