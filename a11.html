Insertion in a Singly Linked List

A complete breakdown of inserting nodes into a linked list in C. We will cover three scenarios: inserting at the beginning, at the end, and after a specific position. Just scroll down to follow the lesson sequentially.

1

The Setup & The Node Structure

Before we insert anything, we must define what a "Node" looks like in C and set up our global pointers.


struct node {
    int data;           // The actual value stored
    struct node *next;  // Pointer to the next node's address
};

// Global pointers needed for our operations
struct node *head = NULL; 
struct node *newnode;

💡 The Golden Rule of `head`:

The head pointer always points to the first node. Never use head to traverse the list! If you change the value of head to move forward, you will permanently lose the starting address of your linked list.

Creating the New Node (The Common Step)

Regardless of where you are inserting the node, you always have to allocate memory for it dynamically and ask the user for data.


// 1. Allocate memory dynamically (returns void pointer, so we typecast)
newnode = (struct node *)malloc(sizeof(struct node));

// 2. Ask user for data and store it using the arrow operator (->)
printf("Enter data you want to insert: ");
scanf("%d", &newnode->data);

For our visualizations below, let's assume malloc allocates memory at address 500, and the user enters the data 2.

Case 1: Insert at Beginning

We want to make our newly created node the very first node in the list.

⚠️ Warning: Order of Operations Matters!

You must connect the new node to the list BEFORE you update the head pointer. If you update head first, you lose the address of the original list!


// Step 1: New node points to the current first node
newnode->next = head; 

// Step 2: Head is updated to point to the new node
head = newnode;

Case 2: Insert at End

To attach a node to the very end, its next pointer must be set to 0 (NULL). Because we only have the head, we must traverse the entire list to find the last node.

💡 Traversal Pointer:

We create a temporary pointer (temp) and set it equal to head. We use temp to travel through the list. We stop traversing when temp->next is 0 (meaning we are standing ON the last node).


struct node *temp;

// Step 1: Because this is the new last node, its next is NULL
newnode->next = 0; 

// Step 2: Start traversing from the beginning
temp = head;

// Step 3: Traverse until temp points to the currently last node
while (temp->next != 0) {
    temp = temp->next;
}

// Step 4: Link the old last node to our new node
temp->next = newnode;

Case 3: Insert After a Location

We want to insert the new node immediately after a specific position provided by the user (e.g., after position 2). We must traverse up to that position using a counter variable (i).

⚠️ Warning: Right Link First!

When splicing a node into the middle of a list, you must make the Right Link (new node pointing to the rest of the list) before the Left Link (previous node pointing to new node). If you do the Left Link first, the rest of the list is severed and lost.


int pos, i = 1;
struct node *temp;

printf("Enter the position:");
scanf("%d", &pos); // Assume user enters 2

// Edge case: Check if position is valid (assuming 'count' holds total length)
if (pos > count) {
    printf("Invalid Position");
} else {
    temp = head;
    
    // Traverse until temp sits ON the target position node
    while (i < pos) {
        temp = temp->next;
        i++;
    }
    
    // Step 1: RIGHT LINK (newnode -> next node)
    newnode->next = temp->next;
    
    // Step 2: LEFT LINK (current node -> newnode)
    temp->next = newnode;
}
