Data Structures: Deque using Circular Array

1. Core Concepts

What is a Deque?
A Deque (Double-Ended Queue) is a generalized queue where insertion and deletion can happen at both ends (Front and Rear). Standard queues only allow insertion at the rear and deletion at the front.

Why use a Circular Array instead of a Linear Array?

In a standard linear array, if the `Rear` pointer reaches the last index, the queue is considered "Full". However, if we have deleted elements from the front, the beginning indices are completely empty, resulting in wasted space.

A Circular Array solves this by logically connecting the last index back to the first index (index 0). If `Rear` is at the end and index 0 is free, `Rear` wraps around to 0.

The Wrap-Around Formula (Incrementing):

Pointer = (Pointer + 1) % N

Where N is the size of the array. Example: (4 + 1) % 5 = 0.

2. Global Variables Setup

Before coding the operations, we define the global state of our Deque. When the queue is empty, both Front and Rear pointers are initialized to -1.

// Define the size of the array
#define N 5

int deque[N];       // The circular array
int f = -1;         // Front pointer
int r = -1;         // Rear pointer

3. Operation: Insert Front / Enqueue Front

Normally, queues insert at the rear. Inserting at the front means the Front pointer must move backwards (f--). But what if f == 0? It must wrap around to the end of the array.

Logic Steps:

  1. Check Full: If (f==0 && r==N-1) OR (f == r + 1).
  2. Check Empty: If f == -1 && r == -1, set both to 0.
  3. Wrap Around: If f == 0, move backwards to f = N - 1.
  4. Normal: Decrement f--.
void enqueuefront(int x) {
    // 1. Is the Queue Full?
    if ((f == 0 && r == N - 1) || (f == r + 1)) {
        printf("Queue is full\n");
    }
    // 2. Is it empty?
    else if (f == -1 && r == -1) {
        f = 0;
        r = 0;
        deque[f] = x;
    }
    // 3. Wrap-around case
    else if (f == 0) {
        f = N - 1;
        deque[f] = x;
    }
    // 4. Normal insertion
    else {
        f--;
        deque[f] = x;
    }
}

4. Operation: Insert Rear / Enqueue Rear

This is the standard queue insertion. We move the Rear pointer forwards. If r == N - 1, it wraps around to 0.

void enqueuerear(int x) {
    // 1. Is the Queue Full?
    if ((f == 0 && r == N - 1) || (f == r + 1)) {
        printf("Queue is full\n");
    }
    // 2. Is it empty?
    else if (f == -1 && r == -1) {
        f = 0;
        r = 0;
        deque[r] = x;
    }
    // 3. Wrap-around case
    else if (r == N - 1) {
        r = 0;
        deque[r] = x;
    }
    // 4. Normal insertion
    else {
        r++;  // or r = (r + 1) % N;
        deque[r] = x;
    }
}

5. Operation: Delete Front / Dequeue Front

Standard queue deletion. We move the Front pointer forwards (f++) to effectively ignore the deleted element. If f == N - 1, it wraps around to 0.

void dequeuefront() {
    // 1. Is it empty?
    if (f == -1 && r == -1) {
        printf("Queue is empty\n");
    }
    // 2. Only one element left?
    else if (f == r) {
        printf("Deleted: %d\n", deque[f]);
        f = -1;
        r = -1;
    }
    // 3. Wrap-around case
    else if (f == N - 1) {
        printf("Deleted: %d\n", deque[f]);
        f = 0;
    }
    // 4. Normal deletion
    else {
        printf("Deleted: %d\n", deque[f]);
        f++;
    }
}

6. Operation: Delete Rear / Dequeue Rear

Deleting from the rear means moving the Rear pointer backwards (r--). If r == 0, wrapping backwards means setting r = N - 1.

void dequeuerear() {
    // 1. Is it empty?
    if (f == -1 && r == -1) {
        printf("Queue is empty\n");
    }
    // 2. Only one element left?
    else if (f == r) {
        printf("Deleted: %d\n", deque[r]);
        f = -1;
        r = -1;
    }
    // 3. Wrap-around case (Moving backwards from index 0)
    else if (r == 0) {
        printf("Deleted: %d\n", deque[r]);
        r = N - 1;
    }
    // 4. Normal deletion
    else {
        printf("Deleted: %d\n", deque[r]);
        r--;
    }
}

7. Utility: Displaying the Deque

Crucial Note: Because the array is circular, a standard for (int i = f; i <= r; i++) loop will fail if Front has wrapped around and is mathematically greater than Rear (e.g., f=4, r=1). We must use a while loop with Modulo Arithmetic.

void display() {
    if (f == -1 && r == -1) {
        printf("Queue is empty\n");
        return;
    }
    
    int i = f;
    // Loop until i reaches r
    while (i != r) {
        printf("%d ", deque[i]);
        i = (i + 1) % N; // Moves forward circularly
    }
    // Print the last element (where i == r)
    printf("%d\n", deque[r]);
}
