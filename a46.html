Implementation of Queue using Stacks

Comprehensive video breakdown & study notes

🧠 1. The Core Problem

To implement a Queue using Stacks, we must understand the conflicting rules of both data structures:

  • Queue (FIFO): First In, First Out. Insertions happen at the Rear, deletions at the Front.
  • Stack (LIFO): Last In, First Out. Insertions (Push) and deletions (Pop) happen at only one end: the Top.

The Challenge: A single stack forces LIFO behavior. To simulate FIFO behavior, we must use two stacks (Stack 1 and Stack 2) to flip the order of the elements.

⚖️ 2. The Two Approaches

The instructor highlights two primary ways to solve this. You trade off time complexity between the Enqueue and Dequeue operations.

  • Method 1 (Focus of this feed): Enqueue is O(1) (Fast) | Dequeue is O(n) (Slow/Costly).
  • Method 2 (Next video): Enqueue is O(n) (Slow/Costly) | Dequeue is O(1) (Fast).

💻 4. C-Code: Global Variables & Setup

Here is the exact code setup from the whiteboard. We define the capacity, initialize two arrays to act as stacks, and maintain a count variable to track elements.

#define N 5
int S1[N], S2[N];   // The two arrays simulating stacks
int top1 = -1;      // Top pointer for Stack 1
int top2 = -1;      // Top pointer for Stack 2
int count = 0;      // Tracks total elements in the queue

Note: We need standard stack helper functions: push1(), pop1(), push2(), and pop2(). These are standard implementations that increment/decrement the respective top pointers.

🚀 5. C-Code: The Enqueue Function (O(1))

Because we are using Method 1, inserting an element is extremely fast. We simply push it onto Stack 1 and increment the element count.

void enqueue(int x) {
    push1(x);       // Push data to Stack 1
    count++;        // Increment total elements tracker
}

⏳ 6. C-Code: The Dequeue Function O(n)

This is the heavy lifter. To remove the "oldest" element, we move everything out of Stack 1 into Stack 2, pop Stack 2, and then restore Stack 1.

void dequeue() {
    int i, a, b;
    
    // 1. Check for Underflow
    if (top1 == -1 && top2 == -1) {
        printf("Queue is empty\n");
        return;
    }
    
    // 2. Move all elements from S1 to S2
    for (i = 0; i < count; i++) {
        a = pop1();
        push2(a);
    }
    
    // 3. Pop the target element from S2
    b = pop2();
    printf("Dequeued element is %d\n", b);
    
    // 4. Update the element count
    count--;
    
    // 5. Move remaining elements back from S2 to S1
    for (i = 0; i < count; i++) {
        a = pop2();
        push1(a);
    }
}
