Doubly Circular Linked List (DCLL)

🧠 1. The Core Concept

Before writing a single line of code, we must understand the two defining rules of a Doubly Circular Linked List.

  • Doubly: Every node remembers who is behind it (prev) and who is in front of it (next).
  • Circular: There are no NULL pointers. The list loops infinitely. The last node's next points to the first node, and the first node's prev points to the last node.

🏗️ 2. Defining the Node Structure

We must tell the C compiler exactly what one of these "blocks" looks like. We do this globally, outside of our functions.

struct node {
    int data;
    struct node *next;
    struct node *prev;
};

// Global pointers to keep track of our list
struct node *head, *tail;
🎙️ Instructor Audio Note: Memory Allocation "How much memory does this node take? The int data takes 4 bytes. The two pointers take 4 bytes each (on a 32-bit compiler). So, one complete node is a 12-byte block of memory."

✨ 3. Creating the List (The First Node)

When the list is completely empty (head == 0), creating the first node requires special logic. Because the list is circular, a single node must point to itself.

void createDCLL() {
    struct node *newnode; 
    head = 0; // Initialize head to NULL
    int choice = 1;

    while(choice == 1) {
        // 1. Allocate memory dynamically & Typecast
        newnode = (struct node*)malloc(sizeof(struct node));
        
        printf("Enter data: ");
        scanf("%d", &newnode->data);

                // 2. CHECK IF LIST IS EMPTY
        if(head == 0) {
            head = tail = newnode;
            
            // Node points to ITSELF to maintain circle
            head->next = head; 
            head->prev = head; 
        }
🎙️ Instructor Audio Note: Typecasting Malloc "malloc allocates the memory and returns a generic void pointer. We must typecast it using (struct node*) so the compiler knows how to divide those 12 bytes into our data, next, and prev compartments."

🔗 4. Inserting Subsequent Nodes

If head is not 0, nodes already exist. We must attach the newnode to the end of the list and carefully rewire the circular links. We must make 5 pointer updates.

        else {
            // 1. Old tail's next points to new node
            tail->next = newnode; 
            
            // 2. New node's prev points back to old tail
            newnode->prev = tail; 
            
            // 3. New node's next wraps around to the first node
            newnode->next = head; 
            
            // 4. First node's prev wraps around to the new node
            head->prev = newnode; 
            
            // 5. Update tail to officially be the new node
            tail = newnode; 
        }
        
        printf("Continue? (1/0): ");
        scanf("%d", &choice);
    } // End While
} // End Function
🎙️ Instructor Audio Note: Why do we use a tail pointer? "Why use tail to insert? Because head must always point to the first node. If we move head, we lose our starting point! Keeping a tail pointer gives us direct, instant access to the last node. This makes insertion an O(1) operation."

🖥️ 5. Displaying the List (Traversal)

Traversing a circular list is a trap for beginners. Because there is no NULL pointer, an standard while(temp != NULL) loop will run infinitely!

void display() {
    struct node *temp;
    temp = head; // Start at the beginning

    if(head == 0) {
        printf("List is empty");
    } 
    else {
                // 🛑 Method 1: The Standard While Loop
        while(temp != tail) {
            printf("%d ", temp->data);
            temp = temp->next;
        }
        // We must print the last node outside the loop!
        printf("%d", temp->data); 
    }
}
🎙️ Instructor Audio Note: A Cleaner Alternative "The code above is a bit clunky because we have to manually print the last node outside the loop. If you want cleaner code, use a do-while loop instead! It guarantees the loop runs at least once, and stops perfectly when we loop back to the head." 
                // 🟢 Method 2: The Pro-Tip Do-While Loop
        do {
            printf("%d ", temp->data);
            temp = temp->next;
        } while (temp != head);

✅ Summary Cheat Sheet

Review these golden rules before your exam:

  • A Doubly Circular Linked List has NO NULL pointers anywhere.
  • tail->next will always equal head.
  • head->prev will always equal tail.
  • When inserting the very first node, head, tail, next, and prev all point to that exact same single node.
