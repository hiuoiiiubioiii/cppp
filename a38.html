Expression Conversions

Complete visual study notes: Postfix & Prefix to Infix (Using a Stack)

Part 1: Postfix to Infix Conversion

Operators come after their operands.

ab + ef / *

📝 The Rules for Postfix:

  1. Scan Direction: Left to Right ( → ).
  2. If Operand (Letter): Push it into the stack.
  3. If Operator (Symbol):
    • Pop the top 2 elements.
    • Top element = Variable A. Next element = Variable B.
    • Combine as: (B operator A) *Always add parentheses!*
    • Push the result back to the stack.

Part 2: Prefix to Infix Conversion

Operators come before their operands.

* + ab / ef

📝 The Rules for Prefix (Notice the differences!):

  1. Scan Direction: Right to Left ( ← ). *Different!*
  2. If Operand (Letter): Push it into the stack.
  3. If Operator (Symbol):
    • Pop the top 2 elements.
    • Top element = Variable A. Next element = Variable B.
    • Combine as: (A operator B) *Different Formula!*
    • Push the result back to the stack.
