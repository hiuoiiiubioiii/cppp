Doubly Circular Linked List: Deletion Algorithms

An in-depth breakdown of the instructor's whiteboard lecture, visualized for easy scrolling and perfect study notes.

🧱 Prerequisites: The Node Structure

Before jumping into deletion algorithms, we must understand the underlying structure. A doubly circular linked list node has three parts, and the ends loop back to each other.

struct node {
    int data;
    struct node *next;
    struct node *prev;
};

// Global pointers maintained by the program
struct node *head = 0; 
struct node *tail = 0;
Key Rule: In a circular list, `tail->next` always holds the address of `head`, and `head->prev` always holds the address of `tail`. We must update these specifically when deleting end nodes.

1. Deletion from the Beginning (delfrombeg)

Goal: Remove the very first node (pointed to by head), re-link the tail to the new first node, and free the memory.

📝 Logic & Code

We use a temporary pointer (temp) to hold the node we want to delete so we don't lose its memory address when moving the head.

void delfrombeg() {
    struct node *temp;
    temp = head; // Step 1: Point temp to the first node

    // Check 1: Is the list empty?
    if (head == 0) {
        printf("List is empty");
    } 
    // Check 2: Is there only ONE node in the list?
    else if (head->next == head) { 
        head = 0;
        tail = 0;
        free(temp);
    } 
    // Check 3: Standard deletion (Multiple nodes)
    else {
        head = head->next;     // A. Move head to the 2nd node
        head->prev = tail;     // B. New head's 'prev' points to tail
        tail->next = head;     // C. Tail's 'next' points to new head
        free(temp);            // D. Destroy the old first node
    }
}

2. Deletion from the End (delfromend)

Goal: Remove the very last node (pointed to by tail), re-link the new last node to the head, and free the memory.

📝 Logic & Code

This is the exact mirror logic of deleting from the beginning. Instead of starting at head, we set temp = tail.

void delfromend() {
    struct node *temp;
    temp = tail; // Step 1: Point temp to the last node

    if (head == 0) {
        printf("List is empty");
    } 
    else if (head->next == head) {
        head = 0; tail = 0;
        free(temp);
    } 
    else {
        tail = tail->prev;     // A. Move tail back to 2nd-to-last node
        tail->next = head;     // B. New tail's 'next' loops to head
        head->prev = tail;     // C. Head's 'prev' loops to new tail
        free(temp);            // D. Destroy the old last node
    }
}

3. Deletion from Specific Position (delfrompos)

Goal: Ask the user for a number (e.g., position 3), iterate through the list using a while loop to find it, bypass it, and delete it.

📝 Logic & Code

This requires a helper function getlength() to prevent the user from deleting a position that doesn't exist.

void delfrompos() {
    struct node *temp;
    int pos, i = 1, l;
    
    temp = head; // Start at the beginning
    
    printf("Enter Position: ");
    scanf("%d", &pos);
    
    l = getlength(); // Fetch total total nodes

    if (pos < 1 || pos > l) {
        printf("Invalid position");
    } 
    else if (pos == 1) {
        delfrombeg(); // Reuse existing function
    } 
    else {
        // A. Traverse to the target node
        while (i < pos) {
            temp = temp->next;
            i++;
        }

        // B. Edge Case: Is it the last node?
        if (temp->next == head) { 
            tail = temp->prev;
            tail->next = head;
            head->prev = tail;
            free(temp);
        } 
        // C. Normal Middle Deletion Bypassing
        else {
            // The 'next' of the node BEFORE temp points to the node AFTER temp
            temp->prev->next = temp->next; 
            
            // The 'prev' of the node AFTER temp points to the node BEFORE temp
            temp->next->prev = temp->prev; 
            
            free(temp);
        }
    }
}
Pro-Tip from Instructor: Instead of writing out the logic for the "last node" edge case manually inside delfrompos(), you could technically just write else if (pos == l) { delfromend(); }. However, writing the raw pointer logic is excellent practice for exams and interviews!
