Delete from Beginning

Goal: Remove the very first node and make the second node the new starting point. If we just move head immediately, we lose the address of the first node forever! We must use a temp pointer to hold it safely.

void DeleteFromBeg() {
    struct node *temp;
    
    // Edge case: List is empty
    if (head == 0) { 
        printf("List is empty\n");
    } else {
        temp = head;         // Step 1: temp holds the first node
        head = head->next;   // Step 2: head moves to the second node
        free(temp);          // Step 3: Safely delete the original first node
    }
}

Delete from End

Goal: Remove the last node. Since a singly linked list can only go forward, we cannot jump straight to the end. We must traverse the list using two pointers: temp to reach the last node, and prevnode to trail exactly one node behind it.

void DeleteFromEnd() {
    struct node *temp, *prevnode;
    temp = head;
    
    // Step 1: Traverse the list to find the last & 2nd-to-last nodes
    while (temp->next != 0) { 
        prevnode = temp;      // prevnode catches up to temp
        temp = temp->next;    // temp moves one step forward
    }
    
    // Step 2: Edge Case (Only 1 node exists)
    if (temp == head) {
        head = 0; 
    } else {
        // Step 3: Sever the tie to the last node
        prevnode->next = 0; 
    }
    
    // Step 4: Free the garbage memory
    free(temp);
}

Delete from Specified Position

Goal: User inputs a position (e.g., pos = 3). To delete node 3, node 2 needs to completely bypass node 3 and link directly to node 4.

Crucial Rule: We must stop our loop at pos - 1 so we have access to the node before the deletion target.

void DeleteFromPos() {
    struct node *temp, *nextnode;
    int pos, i = 1;
    
    temp = head;
    printf("Enter Position: ");
    scanf("%d", &pos);
    
    // Step 1: Traverse to the node directly BEFORE the target (pos - 1)
    while (i < pos - 1) {
        temp = temp->next;
        i++;
    }
    
    // Step 2: Target the node to be deleted
    nextnode = temp->next;
    
    // Step 3: Re-link to completely bypass the target node
    temp->next = nextnode->next; 
    
    // Step 4: Free the target node memory
    free(nextnode);
}

⏱️ Time Complexity Summary

From Beginning

O(1)

Best case scenario. No traversal required. We instantly update the head pointer.

From End

O(n)

Worst case scenario. We must start at head and step through every single node to find the end.

From Position

O(n)

Depends on the index inputted, but Big O always evaluates the worst case (deleting near the end).
